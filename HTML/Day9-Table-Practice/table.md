## Tables in HTML

_Table is a collection of rows and columns. It is used to show the data in structured manner._

_In HTML we can create tables using `<table>` tag. It comes up with one attribute as border by which we can add border to the tables._

_Inside table to create heading to add data we have different tags and elements and several attributes will be used inside the tables._

## Tags used inside tables

1. `<thead>`: This tag includes the heading rows of the tables.
