## Tags in HTML

_Tags in HTML is used to add some data on our webpage. It is predefined. It has some inbuilt values to show the data in structured format. It is a building block for our webpages._

_Any text written within angular braces known as tags. There are two types of tags in HTML as follows_

- **Paired Tags**: Tags which has opening tags `<>` as well as closing tags `</>` known as Paired tags. For example: `<h1></h1>`, `<title></title>` etc.

- **Unpaired tags**: This is also known as self-closing tags. Here we can have either opening tag or closing tag. for Example: `<br>`, `<hr/>`

## Attributes in HTML

_Attributes in HTMl are keywords means predefined words generally used to provide some extra information to the tags or Elements. There are two types of Attributes as follows._

- **Global attributes:** This kind of attributes can be used on any tags and elements. such as `title`, `style`, `class`, `id` etc.

- **Element Specific Attributes:** This kind of attributes will be applicable for specific elements or tags only. It can't be used on every tags or elements. Such as: `lang`, `href`, `src`, `type`, `name` etc...

## elements

_Any content written within opening tag as well as closing tag known as elements. such as `<h1>Hii</h1>` in this example `<h1>` is an opening tag, `Hii` is a content and `</h1>` is a closing tag._
