## Lists

_Lists are collection of multiple elements or items either in ordered or unordered way. In HTML we have three types of Lists._

- **Unordered Lists:** This kind of lists does not follow any orders. By default the type of unordered list will be disc. We can change the type of list using type attribute. We can create unordered list usign `<ul>` tag. Inside Unordered list we can add items using `<li>` tag. There are different types of unordered list such as disc which is by default, circle, square and none.

- **Ordered Lists:** These kind of lists follows certain order to show the priority of the items. We can create Ordered lists using `<ol>` tag. By default it follows numeric order. We can change the type of ordered lists using type attribute inside opening `<ul type= "">` tag. There are five types of Ordered list. By default Numeric, Capital alphabets (A), small alphabets (a), Capital Roman Numerals (I), Small Roman Numerals (i).

- **Description Lists:** These kind of lists provides a description for its each and every items. We can create a Description lists using `<dl>` tag inside this we can use `<dt>` tag which holds description term. To write the explanation of this term we use `<dd>` tag which stands for Description data or definition.
